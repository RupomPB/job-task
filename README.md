# 🚀 Fintrixx - Amazon Growth Partner Landing Page

**Fintrixx** হলো একটি আধুনিক এবং রেসপনসিভ ল্যান্ডিং পেজ ডিজাইন, যা অ্যামাজন সেলারদের ব্যবসায়িক প্রবৃদ্ধি এবং বিজ্ঞাপনী কৌশল অপ্টিমাইজ করতে সাহায্য করার জন্য তৈরি করা হয়েছে। এটি সম্পূর্ণ ক্লিন কোড এবং আধুনিক ডিজাইনের সংমিশ্রণ।

---

## 🎨 Project Overview
এই প্রজেক্টটি একটি সার্ভিস-বেজড ল্যান্ডিং পেজ যেখানে অ্যামাজন গ্রোথ পার্টনারদের বিভিন্ন সেবা এবং তাদের সফলতার পরিসংখ্যান তুলে ধরা হয়েছে। এটি "Style Decor Project" এর একটি অংশ।

### Key Sections:
- **Hero Section:** আকর্ষণীয় কল-টু-অ্যাকশন এবং পরিসংখ্যান কার্ড।
- **Market Stats:** গ্লোবাল অ্যামাজন মার্কেট ডাটা ডিসপ্লে।
- **Service Grid:** ফন্ট অসাম আইকন সমৃদ্ধ চার ধরণের সার্ভিস কার্ড।
- **Dark Sections:** "Why Choose Us" এবং "Our Approach" এর জন্য প্রিমিয়াম ডার্ক থিম।
- **Problem Solving:** অ্যামাজন সেলিংয়ের কঠিন বাস্তবতা এবং সমাধানের তুলনা।
- **Booking & Footer:** মিটিং শিডিউলিংয়ের জন্য ক্যালেন্ডার ডিজাইন এবং প্রফেশনাল ফুটার।

---

## 🛠️ Technologies Used
প্রজেক্টটি তৈরি করতে নিচের টেকনোলজিগুলো ব্যবহার করা হয়েছে:
* **HTML5:** পেজের স্ট্রাকচার তৈরির জন্য।
* **CSS3:** কাস্টম ডিজাইন এবং আধুনিক গ্রিড/ফ্লেক্সবক্স লেআউটের জন্য।
* **Font Awesome:** আইকন এবং ভিক্টর গ্রাফিক্সের জন্য।
* **Google Fonts:** টাইপোগ্রাফির জন্য (Inter/Poppins)।

---

## 🚀 How to Run the Project
আপনার লোকাল মেশিনে প্রজেক্টটি রান করতে নিচের ধাপগুলো অনুসরণ করুন:

1.  প্রথমে প্রজেক্ট ফোল্ডারটি আপনার পিসিতে ওপেন করুন।
2.  নিশ্চিত করুন যে আপনার কাছে একটি `assets/images/` ফোল্ডার আছে এবং তাতে প্রয়োজনীয় ছবিগুলো আছে।
3.  `index.html` ফাইলটি যেকোনো ব্রাউজারে (Chrome, Edge, Firefox) ওপেন করুন।
